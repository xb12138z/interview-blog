# 面试博客

> 一边准备面试，一边努力别把自己学烦了。

## 这是什么

这是我的面试准备博客。

说得直白一点，就是我一边刷题，一边补八股，一边做项目，一边碎碎念的时候，顺手留下来的东西。

这里面会有算法题，会有后端八股，会有项目总结，会有面经，也会有一点日记。毕竟准备面试这件事，本来就不只是“学”，还包括怀疑人生、重拾信心、继续硬着头皮往前冲。

## 我为什么要写它

因为我发现，只刷题不太够，只背八股也不太够。

很多东西如果只是看懂了、背过了，转头还是会忘；但如果把它真的整理出来，用自己的话重新讲一遍，味道就不一样了。  
所以我想把这段准备过程留下来，既是给自己做笔记，也是给未来回头看的自己一点交代。

顺便，如果还能帮到同样在准备面试的人，那就更好了。

## 这里会写什么

- 算法：记录我觉得值得反复看的题，而不是单纯堆数量
- 八股：C++、网络、MySQL、Redis、分布式、消息队列、场景题这些后端常见内容
- 项目：围绕自己的 C++ 即时通讯项目做整理和复盘
- 面经：被问了什么、答得怎么样、哪里该补
- 日记：一些技术以外，但确实属于这段路上的东西

## 我希望它长什么样

我不太想把它做成那种一本正经、看起来很厉害、但读起来很无聊的博客。

我更希望这里像一个真的在认真准备面试的人写下来的东西：  
有整理，有思考，有踩坑，有自己的理解，也有一点不那么标准答案的表达。

毕竟学习这件事，有时候不是“看过就会”，而是得先把它折腾成自己的东西。

## 如果你刚好也在准备

那欢迎你来一起看。

如果这里有一两篇内容刚好能帮你理顺思路，那就挺值。  
如果有些地方写得还不够好，那也正常，我本来就是一边学一边改，一边改一边继续往前走。
