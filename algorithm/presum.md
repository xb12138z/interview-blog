# 前缀和

## 前缀和的用途：

前缀和，顾名思义就是计算从数组起点到数组每个数字之间的和。作用是在得到前缀和数组后，可以帮我们O（1）的计算出数组任意两点之间的和

## 560.和为K的子数组

题目：  
给你一个数组nums和一个整数k，请你统计并返回该数组中和为k的子数组个数

题解：  
子数组的和可以视为两个前缀和相减。
```cpp
int SubarraySum(vector<int>& nums , int k){
    unordered_map<int,int> map;
    int res = 0;
    int sum = 0;
    map[0]++;
    for(auto it : nums){
        sum += it;
        if(map.find(sum - k) != map.end()){
            res += map[sum - k];
        }
        map[sum]++;
    }
    return res;
}
```

##  53.最大子数组和

题目：  
给出一个数组，元素有正有负。求子数组中最大的和为多少

题解：  

思路有二：  
思路一：自不必谈，前缀和。我们只需要记录最小的前缀和，然后用当前和减去之前的最小前缀和，即可得出以当前位置为止的子数组的最大值。  
思路二：我们直接遍历整个数组，然后维护一个值pre - 用于体现之前值对以后的影响，但我们要保持pre始终大于等于0.然后不断地向后遍历。【就像人生一样，前半段如果对你有正向的影响，那就自然要保留。但人生前半段对你有负面影响的话不如舍弃，从0开始】
