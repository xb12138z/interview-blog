# 滑动窗口法

## 滑动窗口法的用途：

什么是滑动窗口法？其实就是维持着一个左右指针，右指针不断地试探直到不满足约束条件时，左指针寻找到满足条件的情况，然后右指针继续向后试探。

所以一般滑动窗口解决的问题一般至少有以下几个特点：

1.数据存储是线性的

2.我们是取一个有约束范围的数据
